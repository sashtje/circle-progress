## Circle progress

- Все размеры взяты с округлением и взяты похожие цвета, т.к. макет только в pdf
- Сборщики и минимизация не используются, т.к. их нет в списке допустимых технологий

Реализация блока Progress находится в папке progress. Чтобы использовать его на странице нужно подключить стили **progress/style.css** к **index.html** файлу и импортировать **progress/script.js** в файл со скриптом.

Инициализация блока Progress выглядит следующим образом:

```js
const progress = new Progress(progressContainer);
```

где **progressContainer** - html-элемент - контейнер куда вставляется блок Progress.

После этого становятся доступны 3 метода:

- **setProgress(value)** - для установки прогресса (отвечает за базовое состояние Normal)
- **setAnimation(animationActive)** - для запуска / остановки анимации прогресса (независимое состояние Animated)
- **setVisibility(hidden)** - управляет скрытием / показом прогресса (состояние Hidden)
